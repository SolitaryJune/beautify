入口会依次检查：

1. 小程序登录状态；
2. QQ 绑定状态；
3. 当前字体大文件权限；
4. 百度网盘绑定状态。

按页面显示的按钮处理：

- “去登录”：完成登录后返回。
- “去绑定 QQ”：按现行 QQ 绑定教程完成验证。
- “去绑定百度网盘”：授权后回到本页。
- “重新检查”：刷新前置状态。
- “开始处理”：全部检查通过后进入处理工作区。

大文件处理依赖登录、QQ 和百度网盘三项前置条件。不要直接打开历史网页入口；必须从小程序发起，确保账号和授权状态有效。

支持范围、大小限制和交付细节以当前工作区显示为准。
