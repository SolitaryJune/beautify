入口：**工具 → 管理功能 → 底包功能**。

## 配置工具

页面展示当前套餐名称、总点数、已使用和剩余点数。选择工具会占用页面标出的点数，取消选择会返还。管理员停用或禁止选择的工具不能加入配置。

## 配置适用群

群号来自当前关联 QQ 的权限中心，只显示已经标记为“底包群”的群：

1. 若列表为空，先去权限中心添加群号。
2. 联系管理员确认底包群标记。
3. 返回底包功能，选择适用群。
4. 检查点数和群列表后点击“保存底包配置”。

保存后专区授权仍会按当前套餐、群状态和账号身份校验。点数、可选工具和套餐规则以当前页面显示为准。
