旧作者指南中的功能已按身份拆分：

| 旧教程内容 | 当前入口 |
|---|---|
| 群成员取件、绑定群 | 会员专区 |
| 群文件、卡密、权限、审核 | 工具 → 管理功能 |
| 商品、订单、数据、收益 | 工具 → 创作功能 → 创作者中心 |
| 官方 Bot 群与群文件 | 工具 → 创作功能 → Bot 群管理 |
| QQ 绑定和工具教程 | 视频教程 |

## 迁移步骤

1. 先确认自己是普通用户、打标者、创作者还是管理员。
2. 完成登录和 QQ 绑定。
3. 从上表进入对应分组。
4. 按当前页面重新核对群、文件和权限状态。
5. 停用旧教程中的固定商业说明、审核时间和网页操作流程。

同一账号可能有多个身份，因此可能同时看到多个分组。入口显示和页面校验共同决定实际可用能力。
