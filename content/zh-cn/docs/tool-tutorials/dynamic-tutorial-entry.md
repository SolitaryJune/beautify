## 打开教程

1. 登录小程序并进入“工具”。
2. 在非搜索状态下查看“视频教程专区”。
3. 点击当前账号可见的教程卡片。
4. 也可以进入具体工具，查看页面上的教程入口。

专区完全不显示时，表示当前账号没有同时满足以下条件的教程：

- 工具授权仍有效。
- 工具当前允许进入。
- 权限不是仅展示状态。
- 已配置有效教程链接。

游客默认工具、登录后默认工具或已过期权限不会因为工具名称相似而自动获得教程。

教程打开前会再次检查授权到期时间。导航失败时，小程序可能提供复制链接的替代方式。
